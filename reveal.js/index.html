Lillith and Wood

In this project we created a framework for digital puppeteering performances, using solely the performerâ€™s bare hands. The framework uses Unity as a base, and hand information is captured using a Leap Motion device. The performer uses a gesture interaction system and precise hand movements to manipulate the puppet in a number of different manners. It is then possible for the audience to view the puppet directly on a big screen, or using virtual or augmented reality headsets, which allows rich interactions.

The Tempest is an experiment based on Shakespeare's play combining theatrical performance with live cinema using virtual reality and gaming technologies. It was produced as a proof-of-concept of the VR Kino+Theater platform.

It is a platform that integrates traditional forms of entertainment (theater and cinema), with advanced interactive media, virtual reality and gaming. In this way, it solves the scalability of audience and presentation familiarity, while providing greater flexibility for innovative alternative formats. This platform was implemented in Unity.

ReactToon

ReactToon is an authoring tool for collaborative creation of stories for children. With a library of objects, children can manipulate scenes and characters in a multi-touch table and create a story that the imagination can think of. The 2D version was implemented in Processing(Java) and the 3D version was implemented in Panda 3D (Python).

MUAN

MUAN is a Stop Motion Animation System for Education. It consists of programs to create, edit, manipulate and visualize animations. The software was conceived by ANIMA MUNDI (International Animation Festival of Brazil) and developed by the team of VISGRAF Laboratory, with the sponsorship of IBM.

GP Brasil

The game GP Brasil is an arcade car racing game that was entirely produced based on open source software, using the GNU/Linux operating system and open libraries for manipulating graphics, sounds and physics simulation.
